## Быстрый запуск

1. Установите Docker Desktop для Windows
2. Скопируйте `.env.example` в `.env`
3. Запустите в терминале:
```bash
docker-compose up -d --build
```


## Доступ к API
- Приложение: http://localhost:3001
- Документация API: http://localhost:3001/docs

Введи в роуте login имя пользователя admin чтобы войти в админский аккаунт
